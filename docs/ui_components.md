# Daftar Komponen UI (User Interface)

**Tautan Mockup Desain (Figma/Canva):** 'https://www.figma.com/design/9uhs3hK29wrGbpPYXW4b7T/workspace-nazwa?node-id=186-700&t=V4llCFgGEqAYBSbf-1'

Berikut adalah daftar komponen UI yang dibutuhkan untuk 3 halaman utama:

### 1. Halaman Landing Page
* **Navbar:** Logo aplikasi, menu navigasi (Beranda, Tentang, Kontak), tombol "Masuk".
* **Hero Section:** Judul utama aplikasi "Prediksi Kemiskinan", teks deskripsi singkat, tombol "Mulai Analisis" (Call to Action).
* **Footer:** Informasi hak cipta dan tautan sosial media.

### 2. Halaman Login
* **Form Container:** Kotak form yang terpusat di tengah layar.
* **Input Fields:** Kolom input untuk Email dan Password.
* **Tombol Login:** Tombol submit utama.
* **Tautan Tambahan:** Teks "Lupa Password?" atau "Daftar Akun Baru".

### 3. Halaman Dashboard
* **Sidebar:** Menu samping untuk navigasi ke berbagai fitur (Ringkasan, Data Prediksi, Pengaturan).
* **Header:** Profil pengguna (avatar) dan tombol notifikasi.
* **Cards/Widget:** Kotak-kotak informasi yang menampilkan ringkasan data (misalnya: Total Data, Tingkat Kemiskinan Saat Ini).
* **Tabel Data:** Tabel yang menampilkan daftar hasil prediksi kemiskinan per daerah.
