\# Blueprint \& Desain UI/UX: Project-Prediksi-Kemiskinan



\## Deskripsi Singkat Aplikasi

Project-Prediksi-Kemiskinan adalah sebuah sistem berbasis web yang memanfaatkan model \*Machine Learning\* untuk menganalisis data metrik dan memprediksi tingkat kemiskinan suatu daerah. Sistem ini dirancang untuk mendukung pengambilan keputusan dan intervensi kebijakan secara cepat dan tepat.



\## Panduan Visual \& Mockup UI

Implementasi antarmuka pengguna (UI) secara penuh akan dieksekusi di Figma. 



\* \*\*Prinsip Desain:\*\* Elegan, bersih (\*clean look\*), dan profesional dengan fokus murni pada UI/UX.

\* \*\*Tata Letak:\*\* Wajib menggunakan sistem tata letak berbasis \*grid\* yang rapi untuk menyajikan tabel data dan grafik hasil prediksi agar mudah dibaca.

\* \*\*Catatan Khusus:\*\* Desain harus tetap minimalis. Dilarang keras menggunakan elemen visual usang atau redundan seperti \*navbar\* kapsul atau logo gantung.
